Major Project 1: Inner Reality

Project Parameters

The requirements for this project was to use Adobe Photoshop to create an image in the style of Dadaists, montage artists, and/or augmented reality that will be contained within, or framed around, a photograph of yourself. To develop this image, I thought about an issue I felt strongly about, which was pollution. From there, I worked to find elements that portrayed this issue.

Technical Aspects

To create this image, I used Adobe Photoshop. I employed many of the tools in Photoshop, including working with color grading (hue, saturation, and brightness), organized layers, and color overlays.
