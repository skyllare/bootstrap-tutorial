Major Project #2: Space Travel Poster

Project Parameters

The requirements for this project was to use Adobe Illustrator to create a poster that depicted a fuctional planet destination. To create this poster, I created a hand-drawn draft which I then used to create the image in Illustrator. I found a color pallette to follow as I was making my images.

Technical Aspects

To create this image, I used Adobe Illustrator. I employed many of the tools in Illustrator, including bitmap tracing, vectors, and gradients.
